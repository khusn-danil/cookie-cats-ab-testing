# A/B Testing Analysis for Mobile Game "Cookie Cats"

Репозиторий содержит комплексный статистический анализ результатов А/Б-тестирования в мобильной игре **Cookie Cats**. Цель эксперимента — оценить влияние переноса первых "ворот" (контрольной точки) с 30 на 40 уровень на вовлеченность игроков и показатели удержания (Retention).

## Стек
* **Язык:** Python
* **Библиотеки:** pandas, scipy.stats, seaborn, matplotlib
* **Инфраструктура:** Автоматическая загрузка датасета напрямую через Kaggle API (`kagglehub`)

---


## Основные этапы исследования
1. **Предобработка данных:** Очистка датасета от аномальных выбросов.
2. **Валидация сплитования (SRM-тест):** Проверка качества разделения трафика (выявлен микроперекос p-value = 0.0085).
3. **Анализ вовлеченности (Количество раундов):** Применение непараметрического критерия Манна-Уитни.
4. **Анализ удержания (Retention 1 & 7 дня):** Оценка долей с помощью критерия Хи-квадрат.

## Ключевые инсайты и выводы
* Перенос ворот на 40-й уровень **не увеличил** количество сыгранных раундов (p-value = 0.0511).
* На 7-й день зафиксировано **статистически значимое падение Retention** в целевой группе с 19.02% до 18.20% (p-value = 0.0016).
* **Бизнес-рекомендация:** Отказаться от раскатки изменений и сохранить ворота на 30-м уровне. Раннее ограничение контента создает эффект недосказанности (клиффхэнгер), заставляя пользователей возвращаться в игру дольше.
